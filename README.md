# MyMediaProject - 新媒体运营工作台

## 📖 平台定位

MyMediaProject 是一个专为新媒体运营人员设计的实用工作平台，专注于**内容创作流程管理**、**角色prompt沉淀**和**运营工具集成**。

**核心价值**：让运营人员专注于内容创意，技术工具服务于运营流程。

## 🎯 三大核心模块

### 1. 角色与Prompt管理
- **角色设定**：沉淀各账号的创作人设和风格要求
- **Prompt模板**：记录已验证的高效创作指令
- **流程记录**：跟踪每个账号的创作流程和效果

### 2. 内容创作与沉淀
- **文案生成**：基于角色设定生成针对性内容
- **内容归档**：按账号、平台、时间组织内容
- **效果追踪**：记录发布数据和用户反馈

### 3. 实用工具集成
- **素材下载**：图片、视频等素材的自动化获取
- **数据处理**：运营数据的简单分析和可视化
- **批量操作**：多账号、多平台的批量管理

## 🚀 快速上手

### 第一步：定义账号角色
在 `roles/` 目录下为每个运营账号创建角色文件：
- 明确账号定位和目标受众
- 定义内容风格和创作要求
- 记录有效的prompt模板

### 第二步：开始内容创作
基于角色设定进行内容生成：
```bash
# 使用角色prompt生成内容
# 内容会自动保存到对应账号目录
```

### 第三步：使用实用工具
```bash
# 下载小红书图片素材
python src/tools/download_xhs_images.py
```

## 📊 运营工作流

### 账号管理流程
1. **角色定义** → 创建账号角色文件
2. **内容创作** → 基于角色生成内容
3. **效果评估** → 记录发布数据
4. **优化迭代** → 更新角色设定

### 内容创作流程
1. **话题研究** → 分析热点和用户需求
2. **Prompt调用** → 使用已验证的创作指令
3. **内容生成** → 基于角色风格创作
4. **素材准备** → 使用工具下载所需素材
5. **发布追踪** → 记录效果数据
