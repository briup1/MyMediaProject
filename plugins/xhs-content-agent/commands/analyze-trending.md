# 分析热榜命令

本命令分析各平台热榜数据，提供爆款选题建议。

## 使用方式

```bash
/xhs:analyze-trending
```

可选参数：
- `--platform`：指定平台（xhs/douyin/weibo），默认xhs
- `--category`：筛选分类

## 执行流程

### 第一步：获取热榜数据

调用热榜获取脚本：

```bash
python /home/jarvisren/.claude/plugins/cache/my-plugins/xhs-content-agent/scripts/fetch_trending.py --platform <platform>
```

**注意**：当前热榜接口为预留功能，如果脚本未实现，提示用户提供热榜数据。

### 第二步：分析热门话题

1. 读取热榜数据
2. 按热度排序
3. 筛选适合的内容类型

### 第三步：评估爆款潜力

对每个热门话题进行评估：

- 时效性：25%
- 共鸣度：30%
- 实用性：20%
- 独特性：15%
- 可操作性：10%

### 第四步：匹配人设

1. 询问用户目标人设
2. 评估每个话题与人设的匹配度
3. 过滤掉不匹配的话题

### 第五步：输出选题报告

生成选题分析报告，包含：
- 热门话题列表
- 爆款潜力评分
- 人设匹配度
- 创作建议

## 注意事项

1. **数据时效性**：热榜数据变化快，建议及时获取
2. **接口状态**：当前热榜接口为预留功能
3. **人设匹配**：始终考虑人设定位的一致性
4. **创作难度**：评估自己的创作能力

## 错误处理

- 热榜获取失败：提示检查网络或手动提供数据
- 人设文件缺失：提示创建或选择人设
- 无匹配选题：建议扩大分析范围或调整人设
