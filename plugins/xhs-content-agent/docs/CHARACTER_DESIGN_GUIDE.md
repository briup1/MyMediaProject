# 人物形象设计指南

## 为什么需要统一的人物形象？

在内容创作中，**统一的人物形象**是建立个人IP的关键。当你的受众在多个帖子中看到相同的形象时，会：

1. **建立记忆点**：形象固定，更容易被记住
2. **增强信任感**：一致性传达专业和真诚
3. **形成品牌认知**：形象即品牌，帮助区分于他人

## 形象设计原则

### 1. 真实感 > 完美感

**避免完美**：完美的人物形象是不存在的，会让人感觉虚假和不真实。

**添加不完美特征**：
- 笑起来眼角有小细纹
- 熬夜后会有淡淡的黑眼圈
- 皮肤不是冷白皮，而是有些健康的气色
- 身材不是黄金比例，而是有些真实的小特点

### 2. 符合人设定位

人物形象必须与人设的定位、性格、内容方向保持一致：

| 人设类型 | 形象特点 | 示例 |
|---------|---------|------|
| 温柔治愈型 | 柔和、亲切、有亲和力 | 幼师、心理咨询师 |
| 专业理性型 | 干练、简洁、有质感 | 职场博主、知识分享者 |
| 活泼可爱型 | 可爱、色彩丰富、充满活力 | 学生党、生活方式博主 |
| 独立女性型 | 坚定、自信、有力量 | 创业者、职场女性 |

### 3. 细节具体化

形象描述要具体到可以被AI绘图工具理解和使用：

**❌ 模糊描述**：
```
她很漂亮，笑起来很甜。
```

**✅ 具体描述**：
```
26-28岁的亚洲女性，中等身材有些肉肉的可爱感。鹅蛋脸带一点婴儿肥，笑起来弯弯的杏眼很有亲和力。自然黑棕色的齐肩中长发，柔软地披散着或随意扎个低马尾。
```

## 形象设计要素

### 基础形象

| 要素 | 描述要点 | 示例 |
|------|---------|------|
| 年龄感 | 具体年龄段的感觉 | 26-28岁的感觉 |
| 体型 | 整体身材描述 | 中等身材，有些肉肉的可爱感 |
| 身高 | 大致身高感觉 | 165cm左右，不高不矮 |

### 面部特征

| 要素 | 描述要点 | 示例 |
|------|---------|------|
| 脸型 | 脸型、是否有婴儿肥等 | 鹅蛋脸，有些婴儿肥 |
| 眼睛 | 形状、大小、神态 | 杏眼，笑起来弯弯的，有亲和力 |
| 鼻子 | 大小、形状 | 小巧，鼻尖微微上翘 |
| 嘴巴 | 嘴型、日常表情 | 嘴角自然上扬，经常带着温和的笑意 |
| 皮肤 | 肤色、质感 | 白皙，有些健康的气色，不是冷白皮 |

### 发型

| 要素 | 描述要点 | 示例 |
|------|---------|------|
| 长度 | 具体长度 | 齐肩的中长发 |
| 颜色 | 自然发色 | 自然黑棕色，不是染过的颜色 |
| 质感 | 发质、卷直度 | 有些自然卷，看起来很柔软 |
| 日常造型 | 日常怎么打理 | 随意扎个低马尾，或是披散着很自然 |

### 服装风格

| 要素 | 描述要点 | 示例 |
|------|---------|------|
| 日常穿衣 | 材质、颜色偏好 | 喜欢宽松舒适的棉麻材质，颜色偏大地色系和莫兰迪色 |
| 代表单品 | 具体单品 | 白色衬衫、米色针织开衫、深蓝色直筒裤 |
| 配饰 | 首饰、配饰 | 简单的银色项链，小巧的耳钉 |
| 鞋子 | 常穿鞋型 | 白色帆布鞋或棕色的小皮鞋 |

### 性格气质的外化

| 要素 | 描述要点 | 示例 |
|------|---------|------|
| 整体气质 | 给人的整体感觉 | 温和但有力量，不是那种柔弱的气质，而是内心坚定的感觉 |
| 表情特点 | 日常表情 | 眼神专注而温暖，不会有太夸张的表情 |
| 动作习惯 | 典型动作 | 说话时可能会轻轻点头，听人说话时眼神会专注看着对方 |

### 不完美特征（重要！）

| 类型 | 示例 |
|------|------|
| 面部 | 笑起来眼角会有小细纹 |
| 生活痕迹 | 熬夜后会有淡淡的黑眼圈 |
| 身体 | 不是黄金比例身材，有些真实的小特点 |

## 形象总结模板

将以上要素整合成一段完整的形象描述，用于AI绘图：

```
[人设名称]的标准形象：[年龄感]的[性别]，[体型]。[面部特征详细描述]。[发型详细描述]。穿着[服装详细描述]。[气质特点详细描述]。[不完美特征]，这些不完美细节让[他/她]更真实可亲。
```

## 示例：幼师小安的形象设计

```
幼师小安的标准形象：26-28岁的亚洲女性，中等身材有些肉肉的可爱感。鹅蛋脸带一点婴儿肥，笑起来弯弯的杏眼很有亲和力，小巧的鼻子，嘴角总是带着温和的笑意。自然黑棕色的齐肩中长发，柔软地披散着。穿着宽松舒适的米色针织开衫内搭白色衬衫，深蓝色直筒裤，简单的银色项链点缀。气质温和坚定，眼神专注温暖，不是那种柔弱的感觉而是内心坚强。笑起来眼角有小细纹，这些不完美细节让她更真实可亲。
```

## 如何在插件中使用

### 1. 更新人设文件

在人设文件的 "### 人物形象设计" 部分填写完整的形象描述。

参考模板：`$CLAUDE_PLUGIN_ROOT/personas/示例人设.md`

### 2. 生成图片时

插件会自动读取人设文件中的形象描述，并在所有图片的提示词中使用，确保：

- ✅ 所有图片使用相同的人物形象
- ✅ 形象描述完整且具体
- ✅ 包含不完美特征，更加真实

### 3. 检查一致性

生成完成后，检查：

1. 人物形象是否一致（脸、身材、发型、穿着）
2. 风格是否统一（色调、光影、构图）
3. 是否有不完美特征的真实感

## 常见问题

### Q: 形象需要和真人一样吗？

A: 不需要。形象是人设的视觉化表达，可以基于真实但不完全相同。关键是保持一致性。

### Q: 可以修改形象吗？

A: 可以，但建议不要频繁修改。形象一旦建立，改变会让已经建立认知的受众感到困惑。

### Q: 不完美特征会不会不好看？

A: 不会。不完美特征让形象更真实、更有亲和力。完美反而是不真实的。

### Q: 形象描述要写多长？

A: 建议100-200字，包含所有要素且具体到AI可以理解和使用。

### Q: 不同人设可以用同一个形象吗？

A: 不建议。每个人设都应该有独特的形象，以区分不同的内容方向和受众定位。
