# 小红书内容创作技能

本技能提供小红书帖子创作的核心规范和最佳实践。

## 小红书帖子标准结构

一篇完整的小红书帖子应包含以下部分：

### 1. 吸引眼球的标题

标题是用户决定是否点击的关键。优质标题具备以下特征：

- **长度控制**：10-25字为佳，不超过30字
- **符号运用**：适当使用emoji增加视觉吸引力
- **关键词前置**：重要信息放在标题前半部分
- **制造悬念或痛点**：引发用户好奇心或共鸣

### 2. 正文内容

#### 开头（前3行决定留存率）

- 直接切入主题，不要绕弯子
- 用emoji或符号分隔增加可读性
- 可以用问句、感叹句引发情绪

#### 主体内容

- **分段清晰**：每段2-4句话，避免大段文字
- **emoji使用**：每2-3段插入一个相关emoji
- **排版技巧**：
  - 使用 `|` 分隔不同部分
  - 使用 `•` 或 `✦` 作为列表符号
  - 重要内容用 `「」` 或 `【】` 标注

#### 结尾

- 引导互动：点赞、收藏、关注
- 提出问题引发评论
- 使用相关话题标签

### 3. 话题标签

标签使用策略：

- **数量**：3-8个标签为佳
- **类型**：
  - 热门标签：增加曝光
  - 精准标签：触达目标人群
  - 品牌标签：建立个人IP
- **位置**：放在正文最后，每个标签单独一行或空格分隔

## 内容类型特点

### 干货分享类

- 结构化呈现，使用步骤、列表
- 突出实用价值和可操作性
- 配图展示效果或步骤

### 种草推荐类

- 真实体验感，避免过度营销
- 突出产品亮点和使用场景
- 多角度展示（图片、细节、对比）

### 生活记录类

- 真实情感表达
- 细节描写引发共鸣
- 适当的生活哲学或感悟

### 知识科普类

- 简化专业术语
- 图文结合辅助理解
- 提供实用知识点

## 爆款内容要素

### 情绪价值

- **共鸣感**：替用户说出心声
- **获得感**：提供有用信息
- **新鲜感**：新颖观点或角度
- **社交货币**：值得分享的内容

### 视觉呈现

- **封面图**：决定80%的点击率
  - 高清、明亮、有主题
  - 文字简洁突出重点
  - 符合小红书审美

- **配图数量**：3-7张为佳
- **图片顺序**：封面→内容展示→细节特写→总结

## 合规要求

### 内容红线

- 避免敏感话题和违禁词汇
- 不涉及政治、宗教、色情
- 不夸大宣传，虚假承诺
- 尊重知识产权，注明出处

### 平台规则

- 不引导外部交易
- 不发布重复内容
- 不使用违规营销手段
- 遵守社区规范

## 写作技巧

### 标题优化

- 数字化：`5个技巧` vs `几个技巧`
- 痛点化：`别再...` `你必须知道...`
- 具体化：`从0到1` vs `如何学习`
- 场景化：`上班族必备` `学生党收藏`

### 内容节奏

- 开头3秒：抓住注意力
- 中间部分：提供价值
- 结尾部分：引导互动

### 语言风格

- 口语化：像和朋友聊天
- 有温度：真诚不套路
- 有个性：展现独特风格
- 有节奏：长短句结合
